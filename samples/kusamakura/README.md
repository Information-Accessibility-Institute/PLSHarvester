This directory contains a sample EPUB of *Kusamakura* by Natsume Soseki
(public domain, via Aozora Bunko) and a PLS lexicon generated from it
using PLSHarvester.  The EPUB file was downloaded from https://archive.org/details/aozorabunko_00776.

の EPUB と、それから PLSHarvester により生成された PLS 辞書が
含まれています。EPUBファイルはhttps://archive.org/details/aozorabunko_00776からダウンロードしました。
